# 🏨 Hotel_Nicolas

Uma aplicação de linha de comando (CLI) desenvolvida em Dart para fins de gerenciamento de reservas de hotel.

Este projeto serve como uma estrutura base robusta, contendo testes automatizados, análise estática de código padronizada e gerenciamento eficiente de dependências.

---

## 🚀 Funcionalidades

* **Interface de Linha de Comando (CLI):** Ponto de entrada direto e intuitivo via terminal.
* **Organização Modular:** Separação clara entre a lógica executável (`bin/`) e a biblioteca interna (`lib/`).
* **Análise de Código Estrita:** Configurado com as recomendações de boas práticas oficiais do ecossistema Dart (`package:lints`).
* **Cobertura de Testes:** Pronto para testes unitários com o `package:test`.

---

## 🛠️ Tecnologias Utilizadas

* **Linguagem:** [Dart](https://dart.dev/) (SDK `>=3.11.1 <4.0.0`)
* **Principais Pacotes:**
  * `path`: Manipulação de caminhos de arquivos de forma independente de plataforma.
  * `lints`: Padronização e análise estática do código (Dev Dependency).
  * `test`: Framework para testes automatizados (Dev Dependency).

---

## 📁 Estrutura do Projeto

O projeto segue a estrutura padrão recomendada para aplicações Dart:

* 📂 **`bin/`**: Contém o ponto de entrada principal (`main.dart`) que executa a aplicação.
* 📂 **`lib/`**: Guarda toda a lógica de negócios, classes e métodos reaproveitáveis.
* 📂 **`test/`**: Contém os testes automatizados para garantir a estabilidade do código.
