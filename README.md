# 🏨 Hotel_Nicolas

A Command-Line Interface (CLI) application developed in Dart for hotel reservation management.

This project serves as a solid foundation, featuring automated testing, standardized static code analysis, and efficient dependency management.

---

## 🚀 Features

- 🖥️ **Command-Line Interface (CLI):** Simple and intuitive terminal-based application.
- 📦 **Modular Architecture:** Clear separation between the executable entry point (`bin/`) and the reusable library (`lib/`).
- ✅ **Strict Code Analysis:** Configured with the official Dart linting rules (`package:lints`) to ensure clean and maintainable code.
- 🧪 **Automated Testing:** Ready for unit testing using the `package:test` framework.

---

## 🛠️ Technologies Used

- **Language:** [Dart](https://dart.dev/) (SDK `>=3.11.1 <4.0.0`)
- **Main Packages:**
  - `path` – Cross-platform file path manipulation.
  - `lints` – Official Dart static analysis and code style rules (Dev Dependency).
  - `test` – Unit testing framework (Dev Dependency).

---

## 📁 Project Structure

The project follows the standard structure recommended for Dart applications:

- 📂 **`bin/`** – Contains the application's main entry point (`main.dart`).
- 📂 **`lib/`** – Holds the business logic, reusable classes, and core functionality.
- 📂 **`test/`** – Includes automated tests to ensure code quality and reliability.
